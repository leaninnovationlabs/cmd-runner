Simplify Your Development Pipelines

Execute build pipelines locally, with full control and no cloud dependencies.

Example Pipeline Configuration

        
    name: myappbuild
    description: Will support creating and test myapp
    steps:
    - name: Compile
      image: golang:1.15
      commands: |
        echo 'Building myapp...'
        go build -o ./bin/myapp ./cmd/myapp
        
    - name: RunTests
      commands: |
        echo 'Running tests...'
        go test -v ./...

Run the following command to see the magic happen

cmd-runnder steps.yaml --out=build.log
